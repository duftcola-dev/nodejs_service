# Initializing Flask app 

    **installing flask by pip : pip install Flask --upgrade pip
    ** creeate your main file :

        app.py 

    **the basic flask app looks like this : 

        from flask import Flask

        app = Flask(__name__)

        @app.route("/")
        def hello_world():
            return "<p>Hello, World!</p>"

        if __name__ == "__main__":
            app.run(debug=True,host='0.0.0.0',port=5000)
        
    ** in order to initialize the flask app run :

        export FLASK_APP=app
        flask run
